# examples/Sequence/bin/README.md

This directory:

    examples/Sequence/bin/

contains wrapper scripts for building files relevant to sequences.

Sample inputs can be found in:

    examples/Sequence/inputs/

## Wrapper documentation

Usage information relevant to scripts in this directory is contained in homonymically named md files. 

For example, if the wrapper script is 

    Sequence_wrapper_script

then the documentation will be named

    Sequence_wrapper_script.md
